ICE challenges

These are some of the ICE challenges I worked on over the last few weeks. Click on each one to open the page in a new tab.

Prophet Card

I was assigned to create a card showing the Prophet and some things about him. I learned about creating borders and basic styling.

Apostle Spotlight

This assignment was to create a "spotlight" of sorts showcasing the Apostles. I learned about formatting images (and how picky CSS can really be about this)

Favorite Devo Table

We made a table showing our favorite devotional along with a placeholder form to submit one's own. I learned a lot about css and tables here.

Holiday Page

This one was a fun cahllenge where we were all given the same HTML and asked to each give our own CSS to it. I spent a lot of time learning how to make the "frosted glass" effect.

Country Flags

Here I recreated flags using CSS grids. Quite an interesting activity. I learned how to add grid gaps here.

CSS Positioning

This one got into some complex things regarding grids in CSS. The main takeaway I had from this one was getting overlap can be really easy if you do it right.

Image Gallery

This was a unique assignment. We were to optmize the images because the browser was loading 12 30MB images as thumbnails. I learned how to optimize images and create thumbnails.

Signup Form

I was tasked with creating a signup form for a daily scripture. I learned about formatting forms in CSS.
